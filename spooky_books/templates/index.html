Welcome to My Spooky Books Catalogue

This Django application is designed to help manage and explore an collection of spooky books pulled from the Google books API. All books and Authors are stored on a postgres database. You can add new books to the catalogue, update existing entries, or even delete them. You can also search the database by book title, and ISBN.

Tech Stack:

  • Python

  • Django

  • Django Rest Framework

  • PostgreSQL

  • Psycopg2

  • React
